## Data Source

## Topics

### Membership value

Compare purchasing habits (tickets
purchased, booking lead time, preferred genres) of the six main
membership tiers versus non-members.

- Preferred genres:
  + Determine genres, segments, subsegments
    * By type
  + Determine which tiers prefer which genres, segments
    * By volume of tickets purchased for each genre
    * *volume of tickets purchased for each segment/subsegment
- Booking lead time:
  + Determine time difference between transaction date/time and event date/time
- Tickets purchased:
  + Measure volume of tickets purchased per tier type
  
About the [Membership | Edinburgh International Festival](https://www.eif.co.uk/support-us/membership)

### Young Audience Insights

Investigate if Young Musician's Passport
(YMP) members book non-music events.

- Variables: membership type, genre
  + Measure YMP members booking categories except music

### Access & Booking

Examine booking patterns/engagement for
customers registered on the Access Pass scheme.

- Variables: Membership type, ticket count, genre, booking lead time, as segment, subsegment
  + Locate access pass members
  + Determine average lead time
  + volume of tickets booked
  + Genre preferences
  + Segment/subsegment preferences

## Process

1. Code the python scripts to analyse data provided about the topics we interested, then generate the sheets on the [output](https://github.com/AdairLi2504/edinburgh-festivals-data-hackathon-2025/tree/main/output).
2. Use the sheets to generate in the previous step to draw the diagrams.
3. Adjust the diagrams to make them more readable and compose a report.
